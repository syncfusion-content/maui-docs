# .NET MAUI Chart Overview

The **Syncfusion .NET MAUI Chart control (SfCartesianChart)** is a powerful data visualization component used to create **interactive, cross-platform charts** for mobile and desktop applications. It enables developers to present complex data clearly using a wide range of chart types, rich interactivity, and flexible customization options.

Designed for modern applications, the chart control helps you build **insightful dashboards, analytical tools, and real-time data visualizations** with minimal effort.

## Why Use Syncfusion MAUI Charts?

The SfCartesianChart is built to simplify data visualization while maintaining performance and flexibility.

- **Comprehensive charting capabilities** for various use cases  
- Optimized for **large data sets and smooth interactions**  
- Seamless **data binding and dynamic updates**  
- Extensive **visual customization and styling options**  
- Built for **cross-platform .NET MAUI applications**

## Key Features

### Multiple Chart Series Support

The Cartesian chart supports a wide range of series types, each tailored for different visualization scenarios. These series help represent trends, comparisons, and distributions effectively.

*See all supported series [Chart Types](https://help.syncfusion.com/maui/cartesian-charts/column)*

### Flexible Data Binding

Easily bind data using property mapping. The control supports binding to collections and enables dynamic updates for real-time scenarios.

### Interactive Zooming & Panning

Explore large datasets in detail:
- Zoom into specific data ranges  
- Pan across data regions  
- Maintain smooth rendering performance  

### Multi-Series Charting

Render multiple series in a single chart to:
- Compare datasets  
- Analyze trends  
- Combine different visualization styles  

### Rich Customization

Customize the appearance of every chart element:

- Axes, gridlines, and labels  
- Legends and titles  
- Series styles and colors  
- Tooltips and markers  

This allows you to align charts with your application’s design language.

## Supported Chart Types

The SfCartesianChart includes a variety of chart types for different data visualization needs:

- Line charts – visualize trends over time  
- Column & bar charts – compare values across categories  
- Area charts – represent volume and magnitude  
- Scatter charts – display relationships between variables  
- Spline charts – smooth curve-based trend visualization  

*Explore all chart types [Chart Types](https://help.syncfusion.com/maui/cartesian-charts/column)*

# .NET MAUI Avatar View API Reference

The .NET MAUI Avatar View control provides a graphical representation of a user’s image. It allows customization of the view by adding an image, background color, icon, and text.

This page provides a comprehensive reference for all available APIs of the SfAvatarView control.

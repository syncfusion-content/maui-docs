# .NET MAUI Chat Overview (SfChat)

The Syncfusion .NET MAUI Chat control offers a contemporary conversational chatbot experience, allowing for flexible customization of the layout to display conversations between two or more users.

## What You Can Do

- Build chatbot and messaging interfaces  
- Display conversations between two or more users  
- Support rich message content (text, images, cards, etc.)  
- Enhance user experience with typing indicators and suggestions  
- Customize chat layout and appearance  

## Key Capabilities

### Message Types

Support a variety of message formats:

- Text messages for standard conversations  
- Calendar and time picker messages  
- Card-based messages for structured content  
- Image and hyperlink messages 

### Time Break (Message Grouping)

Organize conversations effectively:

- Groups messages based on date and time  
- Improves readability and navigation in long conversations

### Typing Indicator

Provide real-time user feedback:

- Shows when another user is typing  
- Enhances conversational experience  

### Suggestions

Improve interaction flow:

- Displays quick reply options  
- Helps users respond faster  
- Reduces typing effort  

### Load More Messages

Handle long chat histories:

- Loads previous messages on demand  
- Enables smooth scrolling through conversation history  

### Styling & Customization

Customize chat appearance:

- Style incoming and outgoing messages  
- Modify visual elements to match application design

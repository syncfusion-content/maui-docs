# .NET MAUI Maps (SfMaps) Overview

- The [.NET MAUI Maps](https://www.syncfusion.com/maui-controls/maui-maps) control is a powerful data visualization component used to present **geographical and location-based data** in a clear and interactive way.

- It helps you build map-based applications such as **population analysis, sales distribution, weather tracking, and route visualization**, with rich customization and interactivity.

## Control Highlights

- Visualize geographic data using maps with ease  
- Display dynamic data with markers, bubbles, and color mapping  
- Interact with maps using selection, tooltips, and legends  
- Customize map visuals based on business needs  
- Suitable for analytical and location-based applications

## Key Features

- **[Shape layer](https://help.syncfusion.com/maui/maps/getting-started#add-shape-layer-maps-elements)** - Load and render map shapes using GeoJSON or shapefile data sources to represent geographic regions  

- **[Data labels](https://help.syncfusion.com/maui/maps/data-labels)** - Display shape names or identifiers directly on the map for better understanding  

- **[Markers](https://help.syncfusion.com/maui/maps/markers)** - Highlight specific locations using built-in icons or custom UI elements positioned by latitude and longitude  

- **[Bubbles](https://help.syncfusion.com/maui/maps/bubble)** - Represent additional data (e.g., population or metrics) using varying sizes and colors  

- **[Shape selection](https://help.syncfusion.com/maui/maps/selection)** - Select and highlight map regions and trigger actions through events  

- **[Legend](https://help.syncfusion.com/maui/maps/legend)** - Provide context for map data using customizable legends  

- **[Colors](https://help.syncfusion.com/maui/maps/shape)** - Assign colors to shapes based on value ranges or categories  

- **[Tooltip](https://help.syncfusion.com/maui/maps/tooltip)** - Show additional contextual information for shapes, markers, and bubbles

    ![.NET MAUI Maps layer overview.](images/overview/net-maui-maps-overview.png)

## Globalization

- Supports localization for region-based data representation  
- Adapts map data presentation for global applications and datasets  

## Real-World Scenarios

- Population density and demographic visualization  
- Sales or revenue distribution across regions  
- Weather and environmental monitoring dashboards  
- Election and political boundary analysis  

## Related Controls

- **DataGrid** - Display map-related tabular data alongside geographic visualization  
- **Chart** - Present statistical insights derived from map data  
- **ListView** - Show location-based data in list format  
- **Scheduler** - Combine with location-based scheduling scenarios

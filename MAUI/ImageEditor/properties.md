# .NET MAUI ImageEditor API Reference

The Syncfusion .NET MAUI ImageEditor (SfImageEditor) is a very handy tool, which is used to edit an image by annotating with freehand drawing paths, text, and built-in shapes; it also allows to crop and flip an image.


This page provides a comprehensive reference for all available APIs of the SfImageEditor control.

# .NET MAUI Chart Overview 

- Create intuitive and visually appealing **proportional data visualizations** in your **.NET MAUI applications** using the **Syncfusion Circular Chart control (SfCircularChart)**.

- Designed to represent **part-to-whole relationships**, it helps you display categorical and percentage-based data clearly across **mobile and desktop platforms**.

## Control Highlights

- Visualize proportions and percentages effectively  
- Highlight category distribution clearly  
- Enable interactive data exploration  
- Update charts dynamically with live data  
- Customize chart appearance for consistent UI design  

## Key features

- **Rich Chart Types** including [Pie Chart](https://help.syncfusion.com/maui/circular-charts/piechart), [Doughnut Chart](https://help.syncfusion.com/maui/circular-charts/doughnutchart), and [Radial Bar Chart](https://help.syncfusion.com/maui/circular-charts/radialbarchart) for proportional data visualization
- **Interactive Experience** with tooltips, selection, and slice highlighting for better readability
- **Dynamic Data Updates** supporting live refresh, lazy rendering, and smooth UI transitions
- **Data Representation & Legends** for clear category mapping and improved interpretation
- **Visual Customization** with styling options for slices, labels, legends, and tooltips 

## Globalization

- Screen reader support, keyboard navigation, and high‑contrast themes for inclusive access
- Localization with right‑to‑left (RTL) layout, time zone handling, and region‑aware formatting for labels, legends, and tooltips

## Real‑World Scenarios

- Budget allocation and expense distribution dashboards
- Market share analysis for comparing category proportions
- Survey results visualization to highlight percentage‑based outcomes
- Sales performance tracking using category splits and ratios

## Related Controls

- Cartesian Chart for trend analysis and multi‑series comparisons
- Funnel Chart for process stages and pipeline visualization
- Pyramid Chart for hierarchical or comparative data representation
- DataGrid for structured tabular data display and manipulation

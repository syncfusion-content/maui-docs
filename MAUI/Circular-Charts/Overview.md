# .NET MAUI Circular Chart Overview (SfCircularChart)

- Create intuitive and visually appealing **proportional data visualizations** in your **.NET MAUI applications** using the **Syncfusion Circular Chart control (SfCircularChart)**.

- Designed to represent **part-to-whole relationships**, it helps you display categorical and percentage-based data clearly across **mobile and desktop platforms**.

## What You Can Do

- Visualize proportions and percentages effectively  
- Highlight category distribution clearly  
- Enable interactive data exploration  
- Update charts dynamically with live data  
- Customize chart appearance for consistent UI design  

## Key Capabilities

### Circular Chart Types

Choose the right representation for proportional data:

- **Pie charts** for simple part-to-whole relationships  
- **Doughnut charts** for enhanced visual layout with center space  
- **Radial bar charts** for comparing values in a circular format  

Explore chart types:  
[Pie Chart](https://help.syncfusion.com/maui/circular-charts/piechart)
[Doughnut Chart](https://help.syncfusion.com/maui/circular-charts/doughnutchart)  
[Radial Bar Chart](https://help.syncfusion.com/maui/circular-charts/radialbarchart)  

### Interactive Experience

Make charts more engaging and informative:

- Tooltip display for data insights  
- Data point selection  
- Slice explode (highlighting)  
- Improved user interaction for better readability  

### Data Updates

Support dynamic and real-time scenarios:

- Live data updates  
- Efficient rendering with lazy updates  
- Smooth UI refresh during data changes  

### Data Representation & Legends

Improve readability and context:

- Legends to identify categories  
- Clear mapping between data and chart segments  
- Better visual interpretation of datasets  

### Visual Customization

Adapt charts to your application’s design:

- Slice and segment styling  
- Data label formatting  
- Legend positioning and appearance  
- Tooltip styling  

Learn more: [Customization Options](https://help.syncfusion.com/maui/circular-charts/appearance)

## Accessibility and Globalization

- **Accessibility** – SfCircularChart ensures usability for all users with built‑in support for screen readers, keyboard navigation, and high‑contrast themes, making proportional data visualizations more inclusive and readable.

- **Globalization** – The chart is designed to support global audiences by providing localization, right‑to‑left (RTL) layout support, time zone handling, and region‑aware formatting for labels, legends, and tooltips.

## Summary

The **Syncfusion .NET MAUI Circular Chart control** provides a focused solution for visualizing proportional data using pie, doughnut, and radial bar charts. With built-in interactivity, flexible customization, and efficient rendering, it is suitable for building modern data-driven applications that require clear and concise visual representation of categorical data.
